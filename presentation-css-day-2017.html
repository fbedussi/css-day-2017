Code Smells #1 / Undoing

DON'T

h2 {
   font-size: 2em;
   margin-bottom: 0.5em;
   padding-bottom: 0.5em;
   border-bottom: 1px solid #ccc;
}
.no-border {
   padding-bottom: 0;
   border-bottom: none;
}

DO

h2 {
   font-size: 2em;
   margin-bottom: 0.5em;
}

.headline {
   padding-bottom: 0.5em;
   border-bottom: 1px solid #ccc;
}

The multicolumn challenge: accepted!

by Lorena Ramonda

slide - video

6 techniques to get a 3 column layout tested:

  1. Table
  2. Div display: table
  3. Div floated
  4. Flexbox
  5. CSS Grid
  6. CSS Columns

Results: speed

  1. Flexbox ~809ms
  2. Table ~860ms
  3. Div floated ~915ms
  4. CSS Grid ~927ms
  5. Div display: table ~1030ms
  6. CSS Columns ~1448ms

Compatibility

  1. Table > ALL
  2. Div floated > ALL
  3. Div display: table > NO IE 6 & 7
  4. Flexbox > NO IE 6-9
  5. CSS Columns > NO IE 6-9
  6. CSS Grid > NO safari < 10.3 & Android Browser

And the winner is...

Flexbox/CSS Grid
